== Description ==

Simple sidebar widget that lists your latest Geeklist activities - links, cards, contributions. Choose one per widget. Add as many widgets you like.

== Screenshots ==

1. Setup view in WordPress administration panel ( Appearance > Widgets )

== Installation ==

Installation as usual.

1. Unzip and Upload all files to a sub directory in "/wp-content/plugins/".
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Add 'Geeklist' widget to your sidebar via 'Appearance' > 'Widgets' menu in WordPress.
4. Change title of widget to what ever you like.
5. Select type of list you like.
6. Enter your Geeklist application keys (http://geekli.st/applications).
7. Enter number of list items you want to see in widget.
